### 소스 코드

[문제 링크](https://www.acmicpc.net/problem/17478)

```java
import java.io.*;

public class Main {
    private static int N;

    public static void main(String[] args) {
        try (BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
             BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out))
        ) {
            N = Integer.parseInt(br.readLine());

            bw.write("어느 한 컴퓨터공학과 학생이 유명한 교수님을 찾아가 물었다.\n");
            bw.flush();
            repeat(0);
        } catch (IOException e) {
            throw new RuntimeException(e);
        }
    }

    private static void repeat(int n) {
        String repeatString = "____";

        if (n == N) {
            String last
                    = repeatString.repeat(n) + "\"재귀함수가 뭔가요?\"\n"
                    + repeatString.repeat(n) + "\"재귀함수는 자기 자신을 호출하는 함수라네\"\n"
                    + repeatString.repeat(n) + "라고 답변하였지.";
            System.out.println(last);
            return;
        }

        String content
                = repeatString.repeat(n) + "\"재귀함수가 뭔가요?\"\n"
                + repeatString.repeat(n) + "\"잘 들어보게. 옛날옛날 한 산 꼭대기에 이세상 모든 지식을 통달한 선인이 있었어.\n"
                + repeatString.repeat(n) + "마을 사람들은 모두 그 선인에게 수많은 질문을 했고, 모두 지혜롭게 대답해 주었지.\n"
                + repeatString.repeat(n) + "그의 답은 대부분 옳았다고 하네. 그런데 어느 날, 그 선인에게 한 선비가 찾아와서 물었어.\"";
        System.out.println(content);

        repeat(n + 1);

        String answer = repeatString.repeat(n) + "라고 답변하였지.";
        System.out.println(answer);
    }
}
```

---

### 문제 풀이

재귀 함수의 기초를 묻는 질문이다. 다만 반복할 수록 앞에 `____` 이 붙기 때문에 0부터 입력받는 N 까지 숫자가 커지도록 했다.   
그리고 마지막 질문에서는 다른 답변을 하기 때문에 `if (n==N)` 을 사용해서 `return` 조건과 마지막 답변을 만들었다.
